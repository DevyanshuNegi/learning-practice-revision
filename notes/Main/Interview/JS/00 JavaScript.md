
[[Global Execution context]]
The **Global Execution Context (GEC)** is ==the default environment created by the JavaScript engine when a script first starts to run==. It represents the code that is not inside any function and serves as the foundation for your entire program. 

[[Hoisting]] 
JavaScript behavior where the interpreter appears to move declarations of functions, variables, and classes to the top of their scope before the code is executed

[[Call stack and Queues]]
