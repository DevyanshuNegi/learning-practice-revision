
This is pure theory. If a question asks "List the issues in Machine Learning," you just need to rattle off these bullet points:

- **What algorithms perform best for which types of problems?** (No Free Lunch theorem - no one algorithm works for everything).
    
- **How much training data is sufficient?** (Ties into PAC Learning ).
- - **How do we handle noisy data?** (What if the labels are wrong?).
    
- **Curse of Dimensionality:** As you add more features (columns), you need exponentially more data to find meaningful patterns.
